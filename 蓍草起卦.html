蓍草起卦

发表于 2025-11-26 分类于 阅读

用蓍草（shī cǎo）起卦是《易经》最传统、最正宗的起卦方法，也就是所谓的“大衍之数”。相比金钱卦（用铜板摇），蓍草法步骤繁琐，耗时较长，但古人认为这种繁琐的过程正是为了收摄心神，体现诚意，从而达到“感而遂通”的境界。

以下是大衍筮法（蓍草起卦）的详细步骤说明：

一、 准备工作

  1. 器材：50根蓍草（也可用长短粗细一致的竹签、围棋子等替代）。
  2. 环境：安静整洁，内心诚敬。
  3. 记录：纸笔，用于记录每一次得出的数字和画卦。

二、 核心原理：大衍之数五十，其用四十有九

  1. 拿出50根蓍草。
  2. 取出一根，恭敬地放在正前方（或装回盒中），代表“太极”。这根草在整个过程中不再使用。
  3. 剩下的 49根 才是用来运算的，代表“太极生两仪”。

三、 求出一爻（由“三变”组成）

求出一个爻（如初九或六二），需要经过三次同样的操作流程，称为“三变”。

【第一变】

  1. 分二（信手平分）： 将手中的49根蓍草，随意分为左右两堆（象征天地）。
  2. 挂一（象征人）： 从右边那一堆中取出一根，夹在左手的小指和无名指之间。
  3. 揲四（以四数之）：
    • 先数左边那一堆：每4根一组数出来，最后会余下1、2、3或4根（注意：如果正好除尽，余数算作4，不能算0）。将这余下的草夹在左手无名指和中指之间。
    • 再数右边那一堆：同样每4根一组，最后余下1、2、3或4根。将这余下的草夹在左手中指和食指之间。
  4. 归奇（合余数）：
    • 将左手指缝间夹着的三部分草（挂一的那根 + 左边余数 + 右边余数）拿下来合在一起。
    • 检查：这一把草的总数，要么是5，要么是9。如果是别的数字，说明刚才数错了。
    • 将这5根或9根草放在一旁（第一变结束，不再混入下一轮）。

【第二变】

  1. 合并剩余： 将“第一变”中左右两堆剩下的草合起来（总数是49减去5或9，即44或40根）。
  2. 分二： 再次将草随机分为左右两堆。
  3. 挂一： 从右边取一根夹在左手小指缝。
  4. 揲四： 左边数4余几，右边数4余几。
  5. 归奇： 将挂一和两边的余数拿下来。
    • 检查：这一次拿下来的草，总数要么是4，要么是8。
    • 将这4根或8根草放在一旁。

【第三变】

  1. 合并剩余： 将“第二变”剩下的草合起来（总数是44/40减去4/8）。
  2. 分二： 再次随机分左右两堆。
  3. 挂一： 右取一夹左手。
  4. 揲四： 左右各除以4取余数。
  5. 归奇： 将挂一和余数拿下来。
    • 检查：这一次拿下来的草，总数要么是4，要么是8。
    • 将这4根或8根草放在一旁。

四、 确定爻的性质

经过上述“三变”之后，最初的49根蓍草，被拿走了三次（归奇），现在桌上还留下一堆没有被拿走的蓍草。我们需要数一下这堆剩下的草有多少根。

剩下的根数只会有四种情况，分别对应不同的“数”和“象”：

  • 剩 36 根：36 ÷ 4 = 9。此为老阳。符号为 —（旁边记个O，表示动爻）。
  • 剩 32 根：32 ÷ 4 = 8。此为少阴。符号为 **- -**。
  • 剩 28 根：28 ÷ 4 = 7。此为少阳。符号为 —。
  • 剩 24 根：24 ÷ 4 = 6。此为老阴。符号为 **- -**（旁边记个X，表示动爻）。

简单记忆法：
看“三变”中拿走的草的组合（多/少）：

  • 拿走 5+4+4 = 13根（最少），剩36 -> 9（老阳）
  • 拿走 9+8+8 = 25根（最多），剩24 -> 6（老阴）
  • 拿走 5+8+8 或 9+4+8 等组合，剩28 -> 7（少阳）
  • 拿走 5+4+8 或 9+4+4 等组合，剩32 -> 8（少阴）

五、 成卦（重复六次）

上面所有的步骤（一、二、三、四）做完，仅仅是画出了一个爻。
你需要将这个过程重复六次，按照从下往上的顺序画卦：

  1. 第一次操作得出的数为初爻（最底下）。
  2. 第二次操作得出的数为二爻。
  3. ……
  4. 第六次操作得出的数为上爻（最顶上）。

这样就得到了一个完整的六爻卦（本卦）。

六、 解卦（变卦）

根据记录下的数字（6、7、8、9）来解卦：

  • 7（少阳） 和 8（少阴）：是静爻，不变。
  • 9（老阳）：是动爻，阳极生阴，在变卦中变为阴爻（- -）。
  • 6（老阴）：是动爻，阴极生阳，在变卦中变为阳爻（—）。

解断原则（朱子七条）：

  1. 六爻不变：以本卦的卦辞占断。
  2. 一爻变：以本卦变爻的爻辞占断。
  3. 二爻变：以本卦两个变爻的爻辞占断，以上爻为主。
  4. 三爻变：以本卦和变卦的卦辞合占，以本卦卦辞为主。
  5. 四爻变：以变卦中两个不变之爻的爻辞占断，以下爻为主。
  6. 五爻变：以变卦中剩下的那个不变之爻的爻辞占断。
  7. 六爻皆变：乾卦用“用九”，坤卦用“用六”，其他卦用变卦的卦辞占断。

总结流程图

  1. 50根 -> 存1不用 -> 用 49根。
  2. 第一变：分二、挂一、揲四、归奇 -> 剩44或40。
  3. 第二变：分二、挂一、揲四、归奇 -> 剩40、36或32。
  4. 第三变：分二、挂一、揲四、归奇 -> 剩36、32、28或24。
  5. 除以4：得 9、8、7、6。
  6. 画爻：重复6次，由下至上。
