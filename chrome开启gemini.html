chrome开启gemini

发表于 2026-02-03 分类于 技术分享

第一步：修改 Local State 配置文件

定位到 Chrome 的用户数据目录（如 Windows 的 %LOCALAPPDATA%\Google\Chrome\User Data\Local State），关闭浏览器后修改以下三项：

  • variations_permanent_consistency_country: 将 "cn" 改为 "us"。

  • variations_country: 将 "cn" 改为 "us"。

  • is_glic_eligible: 将 false 改为 true。

第二步：修改 Chrome Flags 标志位

在浏览器地址栏输入 chrome://flags，将以下两项设为 Enabled：

  • #glic

  • #glic-side-panel

第三步：系统环境要求

  • 语言设置： Chrome 的首选界面语言必须设置为 English（英语）。

  • 网络环境： 必须使用美国节点的网络环境。
