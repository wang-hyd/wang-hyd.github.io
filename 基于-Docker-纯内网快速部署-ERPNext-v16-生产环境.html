基于 Docker 纯内网快速部署 ERPNext v16 生产环境

发表于 2026-05-25 分类于 技术分享

本指南基于 Frappe 官方 Docker 部署最佳实践（frappe_docker/docs/02-setup/06-setup-examples.md），针对局域网内网服务器/旧笔记本（如 ThinkPad E49l）进行了配置优化。去除外网 HTTPS 证书申请和复杂的路由（Traefik），采用轻量级的 Nginx 纯 HTTP 反向代理方案，适合低配置内网服务器本地测试或内部小规模办公使用。

1. 前提条件

在开始前，请确保服务器上已安装以下基础环境：

  • 操作系统：Ubuntu / Debian 或其他支持 Docker 的 Linux 发行版
  • 核心依赖：Docker Engine 以及 Docker Compose v2（确保 docker compose version 可用）

6. 访问与登录

安装完成后，局域网内的任何设备都可以通过浏览器访问：

  • 访问地址：http://<服务器的内网IP>:8080
  • 默认管理员账号：Administrator （注意：A 必须大写）
